### 🚨 Hybrid Production Incident Lab (FLAGSHIP)
Simulated a real production outage using **Amazon Linux + nginx** to replicate high CPU degradation and service disruption.

- Identified abnormal CPU usage using `top`
- Isolated runaway process via `ps aux`
- Restarted services using `systemctl`
- Validated system recovery using `curl` and service checks

**Skills Demonstrated:**  
Linux • Incident Response • Root Cause Analysis • AWS EC2 • Service Recovery
